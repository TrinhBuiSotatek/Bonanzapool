## Ghi chu trang thai

- **Site:** Portal/site ma feature/UC thuoc ve.
- **Feature ID:** Dinh danh canonical cua feature theo site-map (FM-ADM-*, FM-XB-*, FM-PL-*, FM-OPW-*, FM-WLC-*, FM-EX-*).
- **Folder ID:** ID/ten folder thuc te tren disk (de map disk scan <-> row).
- **Module:** Module / nhom chuc nang.
- **Feature/Use case name:** Ten human-readable tu site-map.
- **In scope?:** `Yes` / `No` / `Need confirm`. qc-dashboard-sync KHONG tu doi gia tri nay; user co the edit thu cong.
- **Files stt:** Chi liet ke cac loai file artifact da TIM THAY tren disk. Format: `Specs: V<N><br>WF: V<N><br>...`. Neu khong co gi: `No files yet`.
- **UC review stt:** Ket qua qc-uc-read. Format done: `<Verdict> v<N> (Score X/100)`. Dang chay: raw status.
- **Scenario design stt:** Ket qua qc-func-scenario-design. Format done: `v<N> generated`. Dang chay: raw status.
- **TC design stt:** Ket qua qc-func-tc-design. Format done: `v<N> generated`. Dang chay: raw status.
