# US-EXBOT-011: Submit and approve HL agent key for ExBot (RETIRED)

## Story Statement

As a **USDC Investor preparing to start an ExBot**,
I want to **submit my Hyperliquid agent key for secure encrypted storage and receive approval before bot start**,
So that **the system can execute HL trades on my behalf without ever storing my master private key**.

**INVEST check:** I✓ N✓ V✓ E✓ S✓ T✓

## Acceptance Criteria

```gherkin
AC-EXBOT-011-1: Happy path — agent key submitted and stored securely
  Given the investor provides their HL agent key via the POOL UI
  When the investor submits the agent key
  Then the system encrypts the key with AES-256-GCM using a per-row DEK
    And wraps the DEK with the Master Key (Cloudflare Secrets Store)
    And stores only encrypted_secret, wrapped_dek, secret_iv, secret_auth_tag, dek_iv in hl_agent_keys
    And sets approval_status='pending'
    And returns a confirmation "Agent key received and encrypted. Awaiting approval."

AC-EXBOT-011-2: Happy path — agent key approved, bot start unlocked
  Given hl_agent_keys.approval_status='pending' for a user
  When the admin approves the agent key via the admin dashboard
  Then approval_status transitions to 'approved'
    And approved_at is recorded
    And the investor can now pass the agent key preflight check when starting an ExBot

AC-EXBOT-011-3: Edge case — expired agent key blocks bot start
  Given hl_agent_keys.expires_at has passed
  When the preflight check runs for a new bot start
  Then the system blocks the start with message "Your HL agent key has expired. Please submit a new one."
    And the existing encrypted key record remains (not deleted)

AC-EXBOT-011-4: Negative path — plain key never appears in logs or DB
  Given a worker decrypts an agent key to sign an HL order
  When the signing operation completes
  Then the plain DEK and plain agent key are destroyed (function-scope only, not persisted)
    And no log entry contains the raw key or raw DEK
    And the hl_agent_keys row still contains only encrypted blobs
```
