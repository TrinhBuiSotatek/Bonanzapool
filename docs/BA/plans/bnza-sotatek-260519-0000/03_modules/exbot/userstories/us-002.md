# US-EXBOT-002: Monitor active ExBot status

## Story Statement

As a **USDC Investor with a running ExBot**,
I want to **view the current status, LP range position, hedge size, and margin health of my bot**,
So that **I can verify the system is operating correctly and take action if manual intervention is needed**.

**INVEST check:** I✓ N✓ V✓ E✓ S✓ T✓

## Acceptance Criteria

```gherkin
AC-EXBOT-002-1: Happy path — active bot status visible
  Given the investor has an ExBot with lifecycle_state='active'
  When the investor opens the ExBot status screen in the POOL UI
  Then the screen displays bot status as "Active"
    And shows LP range (tickLower/tickUpper), current tick, and range state (in/out)
    And shows current hedge size (ETH), target hedge size, and drift %
    And shows margin status (ok / warning / critical)
    And shows last light-check timestamp

AC-EXBOT-002-2: Edge case — bot in SAFE_MODE
  Given the investor's ExBot has status='safe_mode'
  When the investor views the status screen
  Then the screen prominently displays "Safe Mode — No new actions" with the reason
    And shows a banner "Your bot is paused for safety. Our team is investigating."
    And all action buttons (rebalance, close-LP, etc.) are disabled except "Close Bot (emergency)"

AC-EXBOT-002-3: Negative path — bot in cooldown after stop
  Given the investor's ExBot is in lifecycle_state='hedge_stopped_cooldown'
  When the investor views the status screen
  Then the screen displays "Stop Fired — Cooldown (Xh remaining)"
    And explains "The hedge stop was triggered. Your LP position remains. The system will attempt to re-hedge automatically after the cooldown."
    And shows a notification indicator for the stop event
```
